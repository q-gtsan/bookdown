In diesem Kapitel werden die Tweets mit dem Hashtag #Bielefeld respektive #bielefeld untersucht und visualisiert.

  1. Setzt dich mit dem Kapitel auseinander
  2. Such dir einen Hashtag aus und führe die Methoden und Funktionen für diesen Hashtag wie in diesem Kapitel aus

Der Sinn dieser Übung ist es, die Programmiersprache R respektive RStudio kennenzulernen.

7.1 Workplace & Authentifizierung Twitter

An dieser Stelle wird nur auf die Herstellung des virtuellen Workplaces hingewiesen. Des Weiteren benutzen wir nun mithilfe der Funktion ’auth_as(“Meine APP”)’die vereinfachte Authentifikations-Methode; ohne Zugangsinformationen eingeben zu müssen. Zunächst wird das notwendige Paket rtweet eingelesen.

Code 
# Einlesen des notwendigen Packages
library(rtweet)

auth_as("Meine App")

7.2 Tweets-Suche

Wir speichern die Tweets in dem Objekt “Bielefeld_tweets” ab. Die Analyse und die Visualierung basiert auf dem Objekt, der die notwendigen Daten enthält.

Code 
# Abruf von Tweets unter den Hashtags #Bielefeld oder #bielefeld
Bielefeld_tweets <- search_tweets("#Bielefeld OR #bielefeld", n=9000, #Suche nach Tweets #Bielefeld oder #bielefeld, max. 9000 Tweets
                                lang = "de", #Sprache der Tweets: Deutsch
                                include_rts = TRUE) #Retweets sollen inkludiert sein

7.3 Häufigste User der Hashtags #Bielefeld oder #bielefeld

Um die zehn häufigsten User aus dem Datensatz zu erhalten, wird zur Vereinfachung des Programmierens das Paket tidyverse benutzt. Das Paket bietet die Möglichkeit direkt über den Code einen leserfreundlicheren (& besser nachvollziehbaren) Code zu erstellen, ohne viel kommentieren zu müssen. Für die Auflistung der häufigsten User sind die Account-Namen notwendig, die über die Funktion users_data(Bielefeld_tweets) eingeholt werden. Dies wird in dem Objekt tweets_bi abgespeichert.

Code 
#Notwendiges Paket für die pipe " %>% "-Programmierung
library (tidyverse)

tweets_bi <- users_data(Bielefeld_tweets) #u.a. die Daten zum Accout-Namen einholen

tweets_bi %>% #Zugriff auf Daten mit #Bielefeld-Tweets
  count (name) %>%   #Anzahl der Twitter-Nutzer-Namen
  top_n(10) %>%      #Top 10 werden ausgewählt
  arrange(desc(n))   #in absteigender Reihenfolge ausgeben

7.3.1.1 Über eine Tabelle mit dem Paket gt

Um eine ansprechendere Visualisierung zu generieren, nutzen wir das Paket gt. Dafür ist das Paket zu installieren:

Code 
install.packages("gt")

Anschließend wird das Paket eingelesen und die Tabelle wird erstellt. Die Dokumentation zu dem Paket sind optional hier nachzulesen.

Code 
#Notwendiges Paket für gt-Tables
library (gt) 

tweets_bi %>% 
  count (name) %>% 
  top_n(10) %>% 
  arrange(desc(n)) %>% 
  gt() %>%  #Erzeugung der Tabelle
  tab_header ( #"Kopf" der Tabelle layouten
  title = "Häufigste User des Hashtags #Bielefeld oder #bielefeld", #Tabellen-Titel
  subtitle = "Absolute Anzahl") %>% # Untertitel
  cols_label(name = "Twitter-Nutzer-Name", n = "Anzahl der Tweets") %>% #Beschriftung der Spalten
  tab_source_note(
    source_note =  paste("Source: Data collected from TwitterAPI via rtweet-package ",Sys.Date())
    ) #Angabe der Quellen

7.3.1.2 Über ein Diagram mit dem Paket ggplot2

Für diese Visualisierung wird das Paket ggplot2 benötigt, welches zuvorderst eingelesen wird (und wenn notwendig installiert wird).

Code 
install.packages("ggplot2")

Die Erstellung und Ausgabe des Diagramms erfolgt in diesem Abschnitt.

Code 
#notwendiges Paket
library(ggplot2)

#Als Graphik
tweets_bi %>%
  count(name, sort = TRUE) %>% 
  na.omit() %>%
  top_n(10) %>% 
  ggplot(aes(reorder (x =name, n), y = n)) + #Erzeugung eines Diagramms
  geom_col(fill = 'steelblue4') + #Farbe der Balken
  coord_flip() + #Diagramm um 90 Grad gedreht
  labs(x = "Name", #Beschriftung der X-Achse
       y = "Anzahl", #Beschriftung der y-Achse
       title = "Who is tweeting? ", #Diagrammtitel
       subtitle = "under Hashtag #Bielefeld or #Bielefeld", #Untertitel des Diagramms
       caption = paste("Source: Data collected from TwitterAPI via rtweet-package ",Sys.Date())) #Quellenangabe für Diagramm

In der obigen Darstellung werden die Daten in einem Balkendiagramm präsentiert. Es ist auch möglich sich die Anzahl der Tweets pro Tag anzeigen zu lassen. Hier wird das Paket lubridate benötigt, welches installiert und eingelesen wird.

Code 
#notwendiges Paket
install.packages("lubridate")

Im Anschluss werden die notwendigen Daten definiert und in einer Zeitreihe durch Balken dargestellt.

Code 
library (lubridate)
Bielefeld_tweets$date = date(Bielefeld_tweets$created_at) #Abspeicherung einer neuen Variable Datum

Bielefeld_tweets_date =  data.frame(table(Bielefeld_tweets$date)) #Erzeugung eines neuen Datensatz mit Datum und Anzahl der Tweets
colnames(Bielefeld_tweets_date) = c('Datum','Total.Tweets') #Umbenennung der Variablennamen

# Create data viz
ggplot(Bielefeld_tweets_date)+ #unterste Ebene des Diagramm erzeugen
  geom_bar(aes(x = Datum, #Spezifizierung X-Achse
               y = Total.Tweets, #Spezifizierung y-Achse
               fill = I('cornflowerblue')), #Farbe der Balken
           stat = 'identity', #Verwendung der absoluten Werte aus dem Data.Frame
           show.legend = FALSE)+ #keine Legende anzeigen
  geom_hline(yintercept = mean(Bielefeld_tweets_date$Total.Tweets), #horizontale Linie basierend auf dem Mittelwert der Anzahl an Tweets
             col = ('red'), #Farbe rot
             size = 0.5)+ #Dicke der Linie
  geom_text(aes(fontface = 'italic', #kursiver Text
                label = paste('Mittelwert:',
                              ceiling(mean(Bielefeld_tweets_date$Total.Tweets)), #Einfügen des automatisch berechneten Mittelwertes
                              'Tweets pro Tag'),
                x = 1, #Position des Texts auf der X-Achse (erstes Datum)
                y = mean(Bielefeld_tweets_date$Total.Tweets)+20), #Position auf der Y-Achse, +20 über dem ersten Balken
            hjust = 'left', #Textposition links
            size = 4, #Textgröße
            col = ('red') )+ #Textfarbe
  labs(title = 'Anzahl der Tweets pro Tag ', #Diagrammtitel
       subtitle = 'mit dem Hashtag #Bielefeld oder #bielefeld', #Untertitel des Diagramms
       caption = paste("Source: Data collected from TwitterAPI via rtweet-package ",Sys.Date()))+
  xlab('Datum')+ #Beschriftung x-Achse
  ylab('Anzahl der Tweets')+ #Beschriftung y-Achse
theme_bw() #Diagramm-Theme schwarz-weiß
